HTML Logische Textauszeichnung

Im früheren HTML wurde eine physische Textauszeichnung verwendet. Diese diente rein der Optik und ermöglichte es beispielsweise, Elemente zu unterstreichen, oder mit fetter Schrift zu versehen. Mit der Einführung von CSS ist dies überflüssig geworden; CSS läst sich auch deutlich einfacher formatieren und trennt Design vom Inhalt, was dem HTML Markup zu gute kommt.

Im heutigen HTML sollten deshalb nur noch semantische Auszeichnungen verwendet werden, die einer Logik, einer Bedeutung unterliegen. Bei der logischen Textauszeichnung entscheidet der Browser, wie ein Element dargestellt wird. Das Standard-Aussehen der tags kann über CSS formatiert werden, jedoch ist die noch nicht bei allen neuen HTML5-Elementen sinnvoll, da diese nicht von jedem Browser interpretiert werden.

Aus dem HTML4 übernommen:

Ein Beispieltext mit Quellcode und wieder normal

Ein Beispieltext mit einer Definition und wieder normal

Ein Beispieltext mit einer stärkeren Betonung (engl. emphasis) und wieder normal

Ein Beispieltext mit einem wichtigen Inhalt und wieder normal

Ein Beispieltext mit einem kurzen Zitat und wieder normal

Ein Beispieltext mit einer Quelle oder einem Autor und wieder normal

Mit dem tag <blockquote> können (längere) Zitate beschrieben werden. Zusätzlich kann im einleitenden blockquote-tag das Attribut "cite" verwendet werden, um den Autor des Zitats auszugeben. ("cite" ist als tag, sowie als Attribut im HTML vorhanden).

Neue tags, die im HTML5 dazu kommen:

<figure> Dieser tag enthält eine oder mehrere Grafiken, sowie optional eine Beschreibung.

<figcaption> Beschreibung der Grafik(en), welche über den figure-tag definiert wurde(n).

<nav> Dieser tag beschreibt eine Navigation.

<header> Dieser tag beschreibt einen Kopfbereich.

<section> Dieser tag beschreibt einen Abschnitt/Textbereich.

<aside> Dieser tag beschreibt ein Seitenelement/sidebar.

<details> Dieser tag beschreibt Details eines Seitenelements.

<article> Dieser tag beschreibt einen Artikel, z.B. in einem Blog.

<footer> Dieser tag beschreibt einen Fußbereich.
