CSS position static

position:static;
Dies ist die Standard-position eines Elements im normalen Elementfluss.
Der Wert static ist manchmal nützlich für das Überschreiben früherer Formatierungsregeln, welche weiter wererbt wurden.

Hier im Beispile besitzt <div> algemein eine position:absolute , welche weiter wererbt wird.
