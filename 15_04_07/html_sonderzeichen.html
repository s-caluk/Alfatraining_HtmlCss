HTML Sonderzeichen

Im HTML gibt es ein Anzahl von Sonderzeichen, die dazu dienen,bestimmten Text als Klartext auszugeben, welcher ansonsten als HTML-Element interpretiert werden würde.

Eine Überschrift 1.Ordnung wird so geschrieben:<h1> Überschrift erster Ordnung.

Copyright-Zeichnen ©

Ein Gedankenstrich – ist etwas länger , als das normale Minus-Zeichen: -.

Englisch Anführungszeichen: "hello"

Deutsche Anführungszeichen: „ Hallo „
