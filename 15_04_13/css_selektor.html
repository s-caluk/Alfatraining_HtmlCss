CSS Selektor

Typ-Selektor: Wenn als Selektor ein bestimmtes HTML-Element gewählt wird, z.B. h1, h2 oder p, so spricht man von einem Typ-Selektor. Er bezieht sich auf einen bestimmten HTML-Typ.
(Momentan können wir für jeden HTML-Typ nur eine Formatierung verwenden. Erst die Aufteilung in sogenannte Klassen macht es möglich, im CSS unterschiedliche Stile z.B. für Textabsätze zu verwenden.)

Es gibt noch weitere Selektoren im CSS, z.B. den Universal-Selektor, welcher später dazu dient, alle von HHTML automatisch gesetzten Abstände zurück zu setzen, so dass mit eigenen Formatierungen gearbeitet werden kann

der von den Ankern schon bekannte ID-Selektor spielt auch im CSS eine besondere Rolle, denn er ist auch für die Vergabe von Individualformaten zuständig.

Hier im Dokument wurde der HTML-Typ <body> mit hellblauer Hintergrundfarbe formatiert. Der HTML-Typ <strong> bekamen ebenfalls eigene Schrift- und Hintergrundfarben.
