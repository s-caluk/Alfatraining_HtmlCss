CSS verschachteln

Die beiden Listen, <ul> und <ol>, besitzen beide Listenpunkte (li).
Mit einer CSS-Formatierung soll nun erreicht werden, dass Listenpunkte, welche in einer <ul> stehen, mit roter Farbe formatiert werden.

Dies wird durch die Pfad-Schreibtweise am einfachsten erreicht. Listenpunkte, welche in der <ul> stehen, werden in roter Schriftfarbe dargestellt, Listenpunkte in der <ol> sind nicht betroffen.

Hier wieder eine <ul> als Ausgangsbasis.
Die rote Schriftfarbe wird so lange an die Kind-Elemnete weiter vererbt, bis eine neue Formatierung erfolg, welche die Farbe ändert: Ab der 3. <ul> wird eine neue Hintergrundfarbe gegeben, grün für die Liste, violett für die Listenpunkte.

Sämtliche Farben werden so lange weiter vererbt, bis sie von anderen überschrieben werden. Die violette Hintergrundfarbe für Listenpunkte wird ab der 3. Liste an alle weiteren <li> vererbt. Ab der 5. Liste erhält der <li> eine gelbe Schriftfarbe.
