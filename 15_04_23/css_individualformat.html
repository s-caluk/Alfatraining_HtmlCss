CSS Individualformat

Neben den Klassen gibt es sogenannte Individualformate. Sie werden über das Universalattribt id angesprochen.

Ein Individualformat darf im Gegensatz zu Klassen jeweils nur 1x pro HTML-Dokkument angeruf werden.
Das Individualformat wird mit einer Raute geschrieben; der name hinter der Raute ist frei wählbar , jedoch müssen die gleichen Regeln wie bei den Klassen befolgt werden ( keine Sonderzeichen , Leerzeichen , Umlaute, Ziffern am Anfag, usw.)

Invidualformate werden meist dazu verwendet, feste bereich innerhalb einer Webseite zu definieren , z.B um den Platz der Navigation festzulegen, die auf jeder Unterseite wieder an der gleichen Stelle auftaucht.
