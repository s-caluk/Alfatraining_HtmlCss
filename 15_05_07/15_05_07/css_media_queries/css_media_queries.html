CSS media queries

Medienabfragen per Stylesheet

Eine Stärke von CSS liegt darin, dass die Darstellung eines Dokuments für verschiedene Ausgabemedien festgelegt werden kann. Die Zuordnung von style zu einem Medium erfolgt mit Hilfe von Medienabfragen (media queries).

Anleitung für Dreamweaver CC:

Ausgangspunkt ist ein fertig gestaltetes Layout mit externer CSS-Datei. Im Dreamweaver rechts im "CSS Designer" das Plus-Zeichen bei "@medien" klicken, um eine neue Medienabfrage hinzuzufügen.

Im Dialogfeld "Medienabfrage definieren" die erforderlichen Bedingungen auswählen:
media:screen
min-width: 580px
max-width: 1090px

Im Stylesheet wird nun automatisch ein Bereich für die jeweilige Medienabfrage erstellt. Innerhalb der Klammern können die Elemente der Webseite per CSS-Regel an die neue Größe angepasst (und gegebenen Falls auch versteckt) werden.

Achtung: Die media queries funktionieren für den Internet Explorer erst ab der Version 9, d.h. ältere Explorer verwenden generell nur die style-Formatierungen, welche ausserhalb der Klammern für die Medienabfragen stehen.
