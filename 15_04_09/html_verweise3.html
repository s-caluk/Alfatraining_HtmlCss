HTML Verweise 3 – Anker

Mit dem Universal-Attribut id können Anker innerhalb einer Webseite gesetzt werden. Jeder Anker besitzt einen eindeutigen Namen und darf nur 1x pro HTML-Element verwendet werden.

Der Name der id darf frei gewählt werden, darf jedoch keine Sonderzeichen, Leerzeichen und Umlaute enthalten, sowie nicht mit einer Zahl beginnen. Es empfiehlt sich eine kurze Schreibweise, sowie Kleinschreibung; im Notfall können Wörter mit Binde- oder Unterstrich getrennt werden.

Der Anker wird über einen Verweis (Link) erreicht. Hier wird der Browser dazu veranlasst, einen Sprung exakt an die Stelle des Ankers auszuführen, so dass dieser ganz oben im Browserfenster sichtbar wird. Dies funktioniert jedoch nur, wenn nach dem Anker noch genügend Seiteninhalt vorhanden ist, damit der Anker weit genug nach oben rutschen kann.
