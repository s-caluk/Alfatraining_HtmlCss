CSS font-size

Darstellungsgröße der Schrift

  • px Einheit Pixel. Obwohl ein Pixel nicht messbar ist, wird er trotzdem als feste Einheit betrachtet.
  • % Prozent.
  • em Einheit em. Relative Angabe bezogen auf die Schriftgröße des Elements. 1em entspricht 100% (bei uns, Schulung alfatraining, im Browser eingestellt: 16px)
