CSS white-space ( Textumbruch)

Festlegung, wie ein Zeilenumbruch innerhalb von Elementen mit Text erfolgen soll.

normal
automatischer Zeilenumbruch
pre
vormatiert, d.h. Zeilenumbuch wie im Editor eingegeben
nowrap
kein automatischer Zeilenumbruch; Umbruch nur möglich durch <br>

dieser Textabsatz besitz eine Vorformatierung über CSS mit pre dies bedeutet, dass Zeilenumbrücheund Einrückengen so dargestellt werden, wie im Editor eingegeben. Gleichgültig, ob mit der Tab-Taste eingedrück wird, oder mittels Leerzeichen.
