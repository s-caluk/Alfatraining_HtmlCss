HTML Tabellen 1

1.) Einfache Tabelle mit Gitternetzlinien

Kleinst mögliche Tabelle:

Inhalt der Tabellenzelle

Erläuterung:

table:
Leitet eine Tabelle ein
tr:
Leitet eine Tabellenzelle ein (tr = table row)
td:
Leitet eine Datenzelle ein (td = table data)

Das Attribut border="1" im einleitenden table-tag beszimmt, das eine Tabelle sichtbare Gitternetzlinien besitzt. Um eine Tabelle ohne Gitternetzlinien zu erstellen, wird das Attribut (ab HTML 5) komplett weg gelassen.

Mit dem tag caption kann eine Tabellen-Überschrift (für die gesamte Tabelle) angegeben werden.
