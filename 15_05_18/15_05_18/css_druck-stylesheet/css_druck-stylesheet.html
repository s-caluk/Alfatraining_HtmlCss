Für den Audruck von Webseiten ist es oftmals günstig, ein eigenes Stylesheet anzufertigen, welches die Seite für den Ausdruck optimiert darstellt (Format, Hintergrundfarben, Ausblenden von nicht benötigten Elementen, etc.).

Für den Ausdruck wird ein 2. Stylesheet eingebunden und mit dem entsprechenden Medientyp für den Druck (media="print") versehen. So kann der Browser unterscheiden, wann er welches Stylesheet verwenden soll.
Wichtig: Auch das "normale" Stylesheet muss medienspezifisch eingebunden werden, damit eine Unterscheidung erfolgen kann.

Das print-Stylesheet ist nur eine Ergänzung zum normalen Stylesheet und liefert Formatierungen oder Änderungen speziell für die Druckausgabe. Aus diesem Grund wird es nach der style.css notiert.
