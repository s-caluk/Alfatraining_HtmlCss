Favicon-Test

Ein "Favicon" ist ein "favorite icon", das verwendet werden kann, das Lesezeichen (bookmark) einer Webseite mit einem individuellen Zeichen zu versehen.

Das Favicon ist eine kleine Grafik, welche in der Adressleiste des Browsers angezeigt wird, sowie bei als Lesezeichen gespeicherten Webseiten.

Wichtig:

  1. Das Dateiformat des Favicons muss ein sogenanntes Windows-Icon sein: dateiname.ico
  2. Favicons haben standardmässig die Größe von 16 x 16 px, manche Browser können zwar größere Formate darstellen, jedoch nicht alle.
  3. Programme zur Erstellung eines Favicons: Z.B. In Photoshop als plugin ist ein .ico-Format abspeicherbar, IrfanView und viele andere Freeware-Programme. Das Favicon kann etweder aus einer bestehenden Grafik heraus umgewandelt, oder selbst gezeichnet werden.
  4. Das Favicon muss im Wurzelverzeichnis stehen, d.h. auf oberster Ebene, neben der index.html liegen, damit es vom Browser angezeigt werden kann.
  5. Einbindung im head-Bereich der HTML-Datei über folgenden link:
    <link rel="shortcut icon" type="image/x-icon" href="favicon.ico">
