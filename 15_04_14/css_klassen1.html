CSS Klassen 1

Es gibt 2 möglichkeiten, Klassen für HTML-Elemente zu notieren:

  1. Für keinen bestimmten HTML-Typ (freie Klasse)
  2. Für ein bestimmtes HTML-Element

Klassen können jedem HTML-Element im öffenden tag mitgegeben werden. Sie dienen dazu, Elemente voneinander unterscheiden zu können, wenn durch den Typ-Selektor, sowie die Pfad-Schreibweise keine genaue Differenzierung möglich ist.

Klassen werden im CSS durch einen Punkt notiert, danach folgt der Name der Klasse. Dieser Name darf frei gewählt werden, darf jedoch:

  • keine Leerzeichen enthalten
  • keine Umlaute oder Sonderzeichen enthalten
  • nicht mit einer Ziffer, einem Unter- oder Binderstrich beginnen
  • nicht zu lang sein
