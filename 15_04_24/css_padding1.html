CSS padding 1 (Innenabstand)

Padding ist ein erzwungener Leerraum zwischen dem Inhalt eines Element und seinem eigenen Elementrand

padding ist sinnvoll für alle HTML-Elemente, die einen Absatz erzeugen, bzw. einen Block bilden, z.B.: h1-h6, div, ul, ol, dl, li, dt, dd, p, figure,...

Es lassen sich 4 Abstände definieren: padding-top, padding-right, padding-bottom, padding-left, sowie eine allgemeine Angaben padding.

Erlaubtsind numerische Angaben (px,em), % und auto

Hier im Beispiel bekommt der div#container ein padding von 30px nach allen Seiten. Durch diese "Innenauspolsteurung" werden alle im Container liegenden Elemente 30 px vom Rand des Containers abgehalten.

Durch Zugabe von padding wähst ein Element, wenn es die Angabe einer festen Breite oder höhe besitzt. der div#container ist mit einer Breite von 800px definiert und besitzt nach Zugabe von padding nun eine Gesamtbreite von 860px

Besitzt eine Element keine feste Breite, bzw. Höhe, so wird es durch Zugabevon padding nicht vergrößert. Hier im Beispiel an der Klasse p.beispiel zu sehen: Nach Zugabe von padding wird der Text weiter nach innen verdrängt, die box (p) wähst nicht in die Breite.

Nach Zugabe von border vergrößert sich die Gesamtbreite von div#conteiner auf 800px.
