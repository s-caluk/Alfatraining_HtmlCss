Lightbox Anleitung [af]

Version: Slimbox 2.05, HTML5-valide (siehe Schritt 4), andifichtner juli 2013

Schritt 1 - Download

  1. Link zur Webseite des Entwicklers: http://www.digitalia.be/software/slimbox2#download
  2. ZIP-Datei entpacken. Es finden sich folgende Dateien darin:
    example.html
    Einfache Beispiel-Seite mit Standard-Konfiguration der Lightbox
    example.jpg
    Bild, welches in der Beispiel-Seite verwendet wird
    README.txt
    Anleitung und Info (engl.)
    Ordner: css
    Lightbox-Stylesheet und auch die Bilder, die für die Lightbox-Label gebraucht werden (nextlabel, closelabel, loading...)
    Ordner: js
    zusammengefasste JS-Datei (slimbox2.js) für die Funktion der Lightbox.
    Ordner: src
    Source der JS-Datei (slimbox2.js) für die Lightbox, sehr ausführlich, deshalb nur geeignet für die Entwicklung, nicht für die Ausführung.
    Ordner: extra
    Einige Extra-Skripte, die zur Lightbox hinzugefügt werden können.

Schritt 2 - Einbindung

  1. Bereite eine HTML-Datei für die Galerie vor. Für die Vorschau-Bilder (oder Text-Links) eignet sich eine Liste mit innenliegenden a-tag (Struktur: ul - li - a).
    Beim Klick auf eines der Vorschau-Bilder soll die Lightbox aktiviert und das jeweilige Bild in groß gezeigt werden.
  2. Slimbox 2.05 basiert auf dem JQuery Framework. Auf alle zwei Javascript-Dateien muss im Head-Bereich der HTML-Datei verwiesen werden:
    <script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.9.1/jquery.min.js"></script> <script type="text/javascript" src="js/slimbox2.js"></script>
    Alternativ kann das erste Skript auch heruntergeladen werden, wodurch die HTML-Seite auch offline betrachtet werden kann.
    Dazu die HTML-Seite "example.html" im Browser aufrufen und den Quellcode anzeigen lassen.
    Durch einen Klick auf die Adresse der im Head verlinkten JS-Datei (<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.9.1/jquery.min.js"></script>) das Skript öffnen und mit copy & paste in eine leere, im Dreamweaver erstellte JS-Datei hinein kopieren. Diese Datei im Ordner "js" abspeichern (z.B. unter dem Namen: "jquery_1-9-1").
    Auf diese Datei wird nun verlinkt (, der 2. Link ist der gleiche, wie oben):
    <script type="text/javascript" src="js/jquery_1-9-1.js"></script> <script type="text/javascript" src="js/slimbox2.js"></script>
  3. Binde die Slimbox CSS-Datei ein (= Empfehlung, oder ergänze Dein eigenes Stylesheet mit den Slimbox-Styles).
    <link rel="stylesheet" href="css/slimbox2.css" type="text/css" media="screen">
  4. Überprüfe das CSS und stelle sicher, dass die eingebundenen Dateien prevlabel.gif und nextlabel.gif im richtigen Ordner liegen. Schau ausserdem noch nach den Pfaden der Dateien loading.gif und closelabel.gif, die in der slimbox.css Datei verlinkt sind.

Schritt 3 - Aktivierung

  1. Füge ein data-rel="lightbox" Attribut zu jedem Link-Tag hinzu, um Slimbox zu aktivieren. Zum Beispiel:
    <a href="bilder/blume01.jpg" data-rel="lightbox" title="Mein Titel">Bild 01</a>
    Optional kannst Du das title Attribut benutzen, um einen Titel zu hinterlegen.
  2. Wenn Du eine Galerie aus verschiedenen zusammengehörigen Bildern hast, mache es genauso wie im 1. Schritt, füge aber noch einen Galerie-Namen in eckigen Klammern zum data-rel-Attribut hinzu. Jede Galerie bekommt ihren eigenen Namen. Zum Beispiel:
    <a href="images/image-1.jpg" data-rel="lightbox[roadtrip]">Bild 01</a> <a href="bilder/blume02.jpg" data-rel="lightbox[roadtrip]">Bild 02</a> <a href="bilder/blume03.jpg" data-rel="lightbox[roadtrip]">Bild 03</a>
    Die Anzahl der Bilder pro Galerie ist unbegrenzt.

Schritt 4 - Validierung

  1. Die heruntergeladene Lightbox ist noch nicht sofort HTML5-valide.
    Webseite mit Lösung hier: https://www.growse.com/news/comments/making-slimbox2-play-nice-with-html5/
  2. In der herunter geladenen Version der Lightbox müssen 2 Änderungen getätigt werden:
    • Alle <a rel="lightbox" href="...">
      müssen ersetzt werden durch: <a data-rel="lightbox" href="...">
      Dies wurde in Punkt 3 bereits erledigt und ist nur wichtig, wenn die Beispiel-Seite (example.html) direkt übernommen wurde.
    • Ganz unten im JavaScript der Lightbox steht folgender Block:
      if (!/android|iphone|ipod|series60|symbian|windows ce|blackberry/i.test(navigator.userAgent)) { jQuery(function($) { $("a[rel^='lightbox']").slimbox({/* Put custom options here */}, null, function(el) { return (this == el) || ((this.rel.length > 8) && (this.rel == el.rel)); }); }); }
      Dieser muss ersetzt werden durch:
      if (!/android|iphone|ipod|series60|symbian|windows ce|blackberry/i.test(navigator.userAgent)) { jQuery(function($) { $("a[data-rel^='lightbox']").slimbox({/* Put custom options here */}, null, function(el) { return (this == el) || ((this.attributes['data-rel'].value.length > 8) && (this.attributes['data-rel'].value == el.attributes['data-rel'].value)); }); }); }
  3. Jetzt ist die Lightbox auch HTML5-valide. Viel Spaß damit! :-)
