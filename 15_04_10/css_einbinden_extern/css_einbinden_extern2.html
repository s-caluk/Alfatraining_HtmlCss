CSS einbinden – extern

Es gibt 3 verschiedenen Methoden,CSS einzubinden:

  1. Formatierung über ein externes Stylesheet (Vorteil: Alle mit dem Stylesheet verknüpften dateien werden automatisch formatiert. diese Methode ist optimal geeignet für Webseiten-Projekte, da Änderungen schnell und problemlos möglich sind.

  2. Formatierung über einen CSS-Bereich im head-Bereich unserer Datei.
  3. Formatierung von CSS direkt im einleitenden tag eines HTML-Elements.
